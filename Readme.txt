Dangsindaesin (당신대신) 앱 실행 안내

1. 개발 환경
- Android Studio
- Gradle 8.1 이상
- Kotlin 1.9.x 이상
- Android SDK 34 이상
- Firebase Realtime Database 사용

2. 인터넷 연결이 필요합니다 (Firebase 연결용).

---

3. 실행 방법

 3-1. Android Studio로 프로젝트 열기


 3-2. 실행
   - Android Studio 상단의 ▶ 실행 버튼 클릭
   - `Pixel 2` 또는 실제 기기 선택 후 앱 실행

---

4. 주요 기능

- 회원가입 (닉네임 & ID 중복 검사 포함)
- 로그인
- 대신 요청 / 대신할 일 목록 / 진행중인 일
- 요청 수락 → 매칭 처리
- 커뮤니티 글 작성, 댓글 작성, 신고 기능
- 모든 요청/글/댓글에 닉네임 표시

---
5. 회원가입
메인 화면에서 회원가입 버튼을 눌러 회원가입을 한다.
---
6. 로그인
만들어진 계정으로 로그인을 한다.
---
7. 대신할 사람이 필요하면 대신요청을 누른다
8. 대신할일 목록에서 마음에 드는 일이 있으면 대신해준다
---
9.커뮤니티
글을 쓰고 싶으면 글쓰기, 원하는 글에는 댓글 작성이 가능하고
글과 댓글 중 규정을 위반하는 게시물은 신고를 할 수 있다.
